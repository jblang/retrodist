# Red Hat Linux

These configs stage early Red Hat installer media from InfoMagic Linux
Developer's Resource CD-ROM sets.

## InfoMagic variants

- `1.1/infomagic`: RedHat Mother's Day + 0.1 from LDR 1995_08, disc 4.
- `2.1/infomagic`: RedHat 2.1 from LDR 1995_11, disc 2.
- `3.03/infomagic`: Red Hat 3.0.3 from LDR 1996_04, disc 2.
